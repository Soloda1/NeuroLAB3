# ЛАБОРАТОРНАЯ РАБОТА 3: Кластеризация (Diabetes Dataset)

## Цель
Провести кластеризацию пациентов на основе медицинских показателей; освоить методы кластеризации, метрики качества, интерпретацию кластеров и визуализацию результатов.

## Структура проекта
```
notebooks/        — отдельный ноутбук под каждый этап  
data/raw/         — исходные данные (diabetes.csv)  
data/interim/     — промежуточные (например, diabetes_stage1.csv)  
src/              — вспомогательные функции (опционально)
```
